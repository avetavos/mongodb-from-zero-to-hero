## คอร์สนี้ทำงานยังไง

<CardGrid>
  <Card title="Context → Problem → Solution" icon="open-book">
    ทุก pattern นำเสนอด้วยรูปแบบเดียวกัน — สถานการณ์, แรงที่มันต้องคลี่คลาย, ทางออก และข้อดีข้อเสียที่มันนำมา
  </Card>
  <Card title="ไดอะแกรมที่อธิบายชัด" icon="puzzle">
    ไดอะแกรมสถาปัตยกรรม, sequence และ state เป็น Mermaid ที่รองรับธีม — saga, สถานะของ circuit breaker, service mesh, การส่งต่อ trace
  </Card>
  <Card title="ตัวอย่างในสี่ภาษา" icon="seti:default">
    pattern เชิงตรรกะ (Saga, Circuit Breaker, Idempotent Consumer) มาพร้อม TypeScript, Python, Go และ Rust ในแท็บที่ sync กัน; pattern เชิง infra แสดง config จริง
  </Card>
  <Card title="สองภาษา + read mode" icon="translate">
    ภาษาอังกฤษและไทย, แบบทดสอบที่บันทึกความคืบหน้า และโหมดอ่านแบบไร้สิ่งรบกวน
  </Card>
